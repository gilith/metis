What is Metis?

Metis is an automatic theorem prover for first order logic with equality.

Features of Metis

  • Coded in Standard ML (SML), with an emphasis on keeping the code as simple as possible.
    • Can be compiled using MLton to give respectable performance on standard benchmarks.
    • Can be compiled using Moscow ML (this version is integrated as a proof tactic in the HOL4 interactive theorem prover).
  • Reads in problems in the standard .tptp file format of the TPTP problem set.
  • Outputs detailed proofs in TSTP format, where each proof step is one of 6 simple rules.

Project Status

After a period of low activity, Metis is once again being actively developed. Users are encouraged to try out or adapt Metis for their own applications: the latest version is Metis 2.0 (release 20070709). Metis is free software, released under version 2 of the GNU General Public Licence.
